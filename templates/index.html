About This OCR Web Application

A lightweight and efficient OCR tool to extract structured text from PDF and TIFF documents using two advanced OCR engines.

Engine 1: PaddleOCR
Lightweight Version

This application uses the PP-OCRv5 Mobile variant of PaddleOCR â€” a compact and highly optimized OCR pipeline for CPUs. It leverages separate detection and recognition models to extract text along with its spatial coordinates.

Ideal for scanned documents, printed text, and structured layouts
Engine 2: Tesseract OCR
Google Developed

Tesseract is an open-source OCR engine developed by Google. It supports over multiple languages and works well for plain-text extraction. In this application, it's used for general-purpose OCR tasks.

Perfect when layout-specific details like bounding boxes are not required

Choose PaddleOCR for structured document analysis with precise handwritten text detection, or Tesseract for quick text extraction.

Document size should be less then 10 MB
